# Acessibilidade & Internet

---

# O que é acessibilidade

Possibilidade de utilização com segurança e autonomia de espaços, equipamentos, transportes e sistemas por pessoas com deficiência ou mobilidade reduzida.

---

# Porque é importante

- Direitos iguais ao acesso à informação.
- Não excluir um grande número de pessoas.
- Beneficia todo mundo.

---

# Números

Entre 190 milhões de brasileiros, 45,6 milhões (24%) sofrem com algum tipo de deficiência.

- Visual: 35,5 milhões.
- Física: 13,3 milhões.
- Auditiva: 9,7 milhões.
- Intelectual: 2,7 milhões.

Enquanto menos de 1% dos sites são acessíveis no Brasil.

---

# Principais dificuldades

Barreiras enfrentadas por pessoas com deficiência visual:

- Imagens sem descrição;
- Vídeos sem alternativa textual ou sonora;
- Funções que não funcionam pelo teclado ou quando ampliados;
- Links mal descritos;
- Tabelas que não fazem sentido quando lidas linearmente;
- Campos de formulário sem descrição adequada;
- Contraste inadequado de cores, etc;

---

# Principais dificuldades

Barreiras enfrentadas por pessoas com deficiência auditiva, física, intelectual e outros (Idosos, problema com concentração, memória, percepção):

- Parágrafos extensos;
- Utilização de linguagem complexa sem necessidade;
- Uso de imagens piscantes que desviam a atenção do usuário;
- Falta de clareza e consistência na organização das páginas;
- Atividades com limite de tempo;
- Conteúdo sem uma linguagem clara e simples;

# Como tornar a web acessível?

## WAI-ARIA

WAI-ARIA é um padrão proposto pela W3C que define maneiras de tornar o conteúdo de sites mais acessíveis para pessoas com deficiências ou que utilizam algum assistente de navegação, como leitor de tela e teclado.

---

# Como tornar a web acessível?

## WAI-ARIA

- Roles
- Estados
- Propriedades

```html
<span data-href="www.medium.com" tabindex="0" role="link">
  Medium
</span>
```

# Tags do HTML

- _a_ para link.
- _p_ para parágrafo.
- _button_ para botão.
- _ul, ol, li_ para lista.
- _h1, h2, h3, h4, h5, h6_ para títulos,

- _table, thead, tbody, td, tr_ para tabelas.
- _header_ para o cabeçalho.
- _nav_ para navegação.
- _footer_ para o rodapé.
